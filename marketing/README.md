## Tips

1. **Consistentie:** Gebruik dezelfde beschrijving/tagline overal
2. **Keywords:** "AI", "Dutch", "MKB/SMB", "pay per use", "no subscription"
3. **Unique Selling Point:** Nederlandse focus + pay-per-use model
4. **Follow-up:** Check wekelijks voor vragen/reviews
5. **Social Proof:** Vraag eerste gebruikers om reviews op G2/Capterra
